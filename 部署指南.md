# 问卷网站部署指南

## 方式一：部署到 Vercel（推荐）

### 步骤

1. **注册/登录 Vercel**：访问 https://vercel.com ，用 GitHub 登录

2. **导入 GitHub 仓库**：
   - 点击 "New Project" → "Import Git Repository"
   - 选择 `ryankklam/gz-ebike-survey` 仓库
   - Branch 选择 `main`（或 `feature/serverless`）
   - 直接点击 "Deploy"

3. **配置环境变量**（仅 serverless 分支需要）：
   - 部署完成后，进入项目 → **Settings** → **Environment Variables**
   - 点击 "Add New"，逐个添加以下变量：

   | 变量名 | 值 | 说明 |
   |--------|-----|------|
   | `ADMIN_PASSWORD` | （您设置的管理员密码） | 管理员面板验证密码 |
   | `JSONBIN_BIN_ID` | （JSONBin 的 Bin ID） | 云端数据存储ID |
   | `JSONBIN_MASTER_KEY` | （JSONBin 的 Master Key） | 云端写入权限 |
   | `JSONBIN_ACCESS_KEY` | （JSONBin 的 Access Key） | 云端读取权限 |
   | `JWT_SECRET` | （自定义随机字符串） | Token 签名密钥 |

   > 以上敏感信息请勿提交到代码仓库中。

4. **获取链接**：
   - 部署完成后获得链接：`https://gz-ebike-survey.vercel.app`
   - 这个链接可以直接分享给他人填写

### Vercel Serverless 架构说明

使用 `feature/serverless` 分支时，项目包含后端 API：

| 接口 | 方法 | 功能 | 权限 |
|------|------|------|------|
| `/api/auth` | POST | 管理员密码验证，返回临时token | 公开 |
| `/api/submit` | POST | 提交问卷答案到云端 | 公开 |
| `/api/stats` | GET | 获取统计数据 | 需token |
| `/api/export` | GET | 导出原始数据 | 需token |

**安全设计**：
- 所有密钥存储在 Vercel 环境变量中，前端代码不含任何敏感信息
- 管理员 token 使用 HMAC 签名，1小时自动过期
- 云端读写通过后端代理，前端不直接接触 JSONBin

## 方式二：部署到 GitHub Pages（免费）

### 步骤

1. **注册/登录 GitHub**：访问 https://github.com ，用微信或邮箱注册

2. **新建仓库**：
   - 点击右上角 "+" → "New repository"
   - 仓库名填：`gz-ebike-survey`
   - 选择 "Public"（公开）
   - 勾选 "Add a README file"
   - 点击 "Create repository"

3. **上传文件**：
   - 进入仓库页面，点击 "Add file" → "Upload files"
   - 将 `survey.html` 文件拖拽上传
   - 点击 "Commit changes"

4. **开启 GitHub Pages**：
   - 进入仓库的 "Settings"（设置）
   - 左侧找到 "Pages"
   - Source 选择 "Deploy from a branch"
   - Branch 选择 "main"，文件夹选 "/ (root)"
   - 点击 "Save"

5. **获取链接**：
   - 等待1-2分钟后，Pages 页面会显示您的网站链接
   - 格式为：`https://你的用户名.github.io/gz-ebike-survey`

> GitHub Pages 不支持后端 API，仅能使用纯前端版本（main 分支）。

## 方式三：部署到 Netlify（免费）

1. 注册 https://app.netlify.com （可用 GitHub 登录）
2. 点击 "Add new site" → "Import an existing project"
3. 连接 GitHub 仓库并选择 `gz-ebike-survey`
4. 直接点击 "Deploy"

## 分支说明

| 分支 | 说明 | 后端 |
|------|------|------|
| `main` | 纯前端版本，数据存 localStorage + JSONBin 直连 | 无 |
| `feature/serverless` | Vercel Serverless 版本，密钥在服务端 | 有（/api/*） |

## 常见问题

### 微信内无法打开链接？
这是微信对外部链接的安全拦截。解决方案：
1. 点击拦截页面的"申请恢复访问"提交申诉
2. 绑定自定义域名并完成 ICP 备案（彻底解决）
3. 在电脑浏览器或手机 Safari/Chrome 中打开（不受影响）

### 如何切换部署平台？
后端 API 使用统一路径 `/api/*`，前端代码无需修改。如需切换到其他云平台，只需重写对应的后端文件即可。
