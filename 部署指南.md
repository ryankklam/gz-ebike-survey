# 问卷网站部署指南

## 方式一：部署到 GitHub Pages（免费，推荐）

### 步骤

1. **注册/登录 GitHub**：访问 https://github.com ，用微信或邮箱注册

2. **新建仓库**：
   - 点击右上角 "+" → "New repository"
   - 仓库名填：`gz-ebike-survey`
   - 选择 "Public"（公开）
   - 勾选 "Add a README file"
   - 点击 "Create repository"

3. **上传文件**：
   - 进入仓库页面，点击 "Add file" → "Upload files"
   - 将 `survey.html` 文件拖拽上传
   - 点击 "Commit changes"

4. **开启 GitHub Pages**：
   - 进入仓库的 "Settings"（设置）
   - 左侧找到 "Pages"
   - Source 选择 "Deploy from a branch"
   - Branch 选择 "main"，文件夹选 "/ (root)"
   - 点击 "Save"

5. **获取链接**：
   - 等待1-2分钟后，Pages 页面会显示您的网站链接
   - 格式为：`https://你的用户名.github.io/gz-ebike-survey/survey.html`
   - 这个链接可以直接发到微信群，别人点击即可打开填写

## 方式二：部署到 Vercel（免费，速度更快）

1. 注册 https://vercel.com （可用 GitHub 登录）
2. 点击 "New Project" → "Import Git Repository"
3. 选择刚才创建的 GitHub 仓库
4. 直接点击 "Deploy"
5. 部署完成后获得链接，如：`https://gz-ebike-survey.vercel.app/survey.html`

Vercel 在国内访问速度通常比 GitHub Pages 快。

---

## 方式三：部署到 Netlify（免费）

1. 注册 https://app.netlify.com （可用 GitHub 登录）
2. 点击 "Add new site" → "Deploy manually"
3. 将 `survey.html` 拖拽到上传区域
4. 自动生成链接，如：`https://random-name.netlify.app/survey.html`
